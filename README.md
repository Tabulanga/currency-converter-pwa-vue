# currency-converter-pwa-vue
## PWA prototype

Written as a knowledge test
Used Vue, Bootstrap, IndexedDB
Testing by MochaJS

See [Online](https://tabulan.ga/converter/).


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
